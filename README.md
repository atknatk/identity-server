## Description

[Identy-Server] Identy Server.

## Installation

```bash
$ npm install
```

## Migration

```bash
$ npm run typeorm:migration:generate -- my_init
$ npm run typeorm:migration:run
```

## Seed

```bash
$ npm run start:dev:db:seed
```

## Running the app

```bash
# development
$ npm run start

# watch mode
$ npm run start:dev

# production mode
$ npm run start:prod
```

## Test

```bash
# unit tests
$ npm run test

# e2e tests
$ npm run test:e2e

# test coverage
$ npm run test:cov
```
